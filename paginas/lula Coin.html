LULA COIN

DESCRIÇÃO

O Lula Coin é um projeto que visa explorar os fundamentos das criptomoedas, com destaque para os conceitos de blockchain. Desenvolvido utilizando Python como backend e HTML, CSS e JavaScript para o frontend, o protótipo da Lula Coin pretende oferecer uma visão prática e educativa sobre o funcionamento de uma criptomoeda.

Blockchain

A tecnologia blockchain é a espinha dorsal do Lula Coin. Essa abordagem descentralizada e imutável proporciona transparência e segurança nas transações. Cada bloco contém um registro de transações e é vinculado ao bloco anterior por um índice hash, criando uma cadeia de blocos que é resistente a adulterações.

Desenvolvimento

O backend do Lula Coin foi construído em Python utilizando Flask para criar uma API RESTful. A interface do usuário, desenvolvida com HTML, CSS e JavaScript, proporciona uma experiência interativa e amigável. Foram implementados os módulos de cadastro, login, transferência e mineração.
